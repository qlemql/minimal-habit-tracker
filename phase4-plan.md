## 1. iOS 홈/잠금 화면 위젯 (최우선)

### 개요
앱을 열지 않고 위젯에서 바로 습관 체크. "0초 경험"으로 5초 원칙의 궁극적 실현.

### 위젯 종류
| 위젯 | 크기 | 기능 |
|------|------|------|
| 소형 (Small) | 2x2 | 오늘 달성률 원형 그래프 |
| 중형 (Medium) | 4x2 | 습관 3개 + 체크 버튼 |
| 잠금 화면 | Inline | 스트릭 숫자 또는 달성률 |

### 기술 요구사항
- `expo prebuild` 후 네이티브 iOS 타겟 추가
- WidgetKit (SwiftUI) — iOS 17+ Interactive Widget으로 체크 가능
- App Groups로 앱 ↔ 위젯 데이터 공유 (UserDefaults)
- `widgetData.ts`(이미 구현됨) → SharedDefaults에 JSON 기록

### 예상 기간
2주 (SwiftUI 위젯 개발 + 데이터 브리지 + 테스트)

## 2. 스트릭 보상 시스템

### 개요
연속 달성 시 아이콘/색상 언락 → 꾸미는 재미 학습 → 프리미엄 유료 전환 유도

### 보상 테이블
| 연속일수 | 보상 | 무료/유료 | 근거 |
|---------|------|----------|------|
| 3일 | 특별 축하 애니메이션 | 무료 | 초기 이탈 방지 |
| 7일 | 아이콘 1개 | 무료 | 첫 주간 사이클 완성 |
| 14일 | 색상 1개 | 무료 | 의식적 노력기 종료 |
| 21일 | 아이콘 2개 + 색상 1개 | 무료 | 21일 법칙 |
| 30일 | 테마 미리보기 | 열람 무료 / 적용 유료 | 전환점, 구매 욕구 자극 |
| 50일 | 위젯 스킨 미리보기 | 열람 무료 / 적용 유료 | 자동화 단계, 결제 의향 최고 |
| 100일 | 한정 아이콘 "백일장" | 무료 | 핵심 유저 충성도 강화 |
| 365일 | 한정 아이콘 "1주년" | 무료 | 감사 표현 |

### 수익화 퍼널
```
무료 보상으로 "꾸미는 재미" 학습 (1~21일)
    ↓
30일 프리미엄 테마 미리보기 → 욕구 자극
    ↓
구매 선택지:
  - 아이콘 팩 ₩1,900 / 색상 팩 ₩1,900
  - 테마 ₩2,900 / 위젯 스킨 ₩2,900
  - Pro ₩6,500 (전부 포함, 최고 가성비)
```

### UX 원칙
- 메인 화면에 보상 관련 UI 없음
- 새 화면 만들지 않음 (기존 피커에 잠금 상태만 추가)
- 보상 알림은 축하 오버레이 확장으로만
- 내부 테스트에서 5초 원칙 위반 시 즉시 철회

### 예상 기간
2주 개발 + 1주 테스트
