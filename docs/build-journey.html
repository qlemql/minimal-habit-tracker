싹 - 습관 트래커

빌드 & 배포 여정 — 만난 에러와 해결 과정

Phase 1: 위젯 데이터 동기화
1. 위젯이 빈 화면
앱에는 습관이 있지만 위젯에서 "습관을 등록해주세요" 표시. 앱 ↔ 위젯 간 데이터가 전달되지 않는 상태.
2. syncWidgetData() 호출 누락 발견
add.tsx, edit.tsx에서 습관 추가/수정/삭제 후 위젯 동기화를 호출하지 않음.
syncWidgetData() 추가
Error #1
Module exists: false — SharedDefaultsModule not found. NativeModules에서 모듈을 찾을 수 없음
Fix
New Architecture(newArchEnabled: true)에서 구식 NativeModules 브릿지가 작동하지 않음. Expo Modules API로 전환하여 SharedDefaultsModule 재작성.
Error #2
Expo autolinking이 모듈을 인식하지만 Pod가 설치되지 않음. ExpoModulesProvider에 모듈 미등록.
Fix
.podspec 파일과 package.json 누락이 원인. 두 파일 생성 + Config Plugin(withSharedDefaults.js)으로 Podfile에 자동 추가.
Error #3
CocoaPods could not find compatible versions — required a higher minimum deployment target
Fix
Podfile의 최소 대상이 15.1인데 podspec이 16.0을 요구. podspec의 platform을 ios 15.1로 변경.
위젯 데이터 동기화 완료!
ExpoModulesProvider에 SharedDefaultsModule 등록 확인. 앱에서 습관 추가/체크 시 위젯에 즉시 반영.
Phase 2: 브랜딩 전면 개편
3. 앱 아이콘 리디자인
한국 시장 리서치 → 컨셉 6종 제작 → PM 피드백 반영 → "세 잎 새싹" 확정. 뾰족한 잎 형태 + 크림 배경 + 체크마크.
Error #4
스플래시 이미지가 아이콘과 다르게 보임. 뾰족한 잎이 둥글둥글하게 변형됨.
Fix
Expo의 @expo/image-utils가 prebuild 시 이미지를 재처리하며 형태 변형. 스플래시 전용 이미지를 별도 제작 (투명 배경 + 뾰족한 잎).
4. 크림 테마 도입
기본 테마를 dark → cream으로 변경. 배경 #FFF8F0, 세이지 그린 액센트 #5B8C6A, 브라운 톤 텍스트. 위젯/축하 오버레이/빈 상태도 테마 대응.
5. 네이밍 통일
Minimal Habit Tracker → 싹 - 습관 트래커. 앱 내부, README, CLAUDE.md, 스토어 메타데이터, 랜딩 페이지, 개인정보처리방침 등 전체 파일 교체.
Phase 3: 스토어 출시 준비
6. 법적 문서 작성
개인정보처리방침 + 이용약관 HTML 작성. 설정 화면에 "법적 정보" 섹션 추가 (외부 링크).
7. Cloudflare Pages 배포
ssak-habit-tracker.pages.dev에 법적 문서 호스팅. 무료 HTTPS, 로그인 없이 공개 접근 가능.
8. Android 권한 정리
POST_NOTIFICATIONS, SCHEDULE_EXACT_ALARM만 허용. 카메라/위치/마이크/파일 접근 명시적 차단(blockedPermissions).
Phase 4: EAS Build
9. Apple Developer 가입
$99 결제 → 48시간 대기 → 활성화 완료.
Error #5
Authentication with Apple Developer Portal failed! An unexpected error occurred.
Fix
계정 활성화 직후 Apple 시스템 전파 지연. 시간 경과 후 재시도하여 인증 성공.
Error #6
기기 등록 시 "보안 지연" 발생. 1시간 대기 필요.
Fix
Apple 보안 정책에 의한 대기. 1시간 후 기기 등록 완료.
Error #7
Build failed: Provisioning profile "*[expo] com.qlemql.minimalhabittracker.widget AdHoc" doesn't support the group.com.qlemql.minimalhabittracker App Group.
Fix
위젯 프로비저닝 프로파일에 App Group 권한이 누락됨. Apple Developer Portal에서 App Group 등록 → 위젯 App ID에 연결 → 기존 프로파일 삭제 후 재생성.
10. 빌드 재시도 (현재)
새 프로비저닝 프로파일로 EAS Build 진행 중. 성공 시 실기기 테스트 → App Store 제출.
