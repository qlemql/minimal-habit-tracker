## 2. 스토어 등록정보

### 짧은 설명 (80자 제한)
```
딱 3개 습관만. 5초면 끝. 하루 쉬어도 흐름은 이어져요. 작은 습관이 자라는 곳.
```

### 자세한 설명 (4000자 제한)
```
습관 앱이 너무 복잡하고 지치셨나요?

싹은 다릅니다.

🌱 딱 3개 습관
습관을 무제한으로 추가할 수 없어요. 의도적 제약입니다. 정말 중요한 3개만 심어보세요.

⚡ 5초면 끝
앱 열고, 탭 한 번, 닫으세요. 그게 전부입니다.

🌿 이어가기
다른 앱은 하루 빠지면 스트릭이 0이 됩니다. 싹에서는 하루 쉬어도 흐름이 이어져요. 연구에 따르면, 하루 빠지는 것은 습관 형성에 거의 영향이 없습니다.

✨ 꾸준함에 보상
흐름을 이어가면 새로운 아이콘과 색상이 해금돼요. 꾸준히 키운 만큼 자라나는 보상.

📋 더 많은 기능
• 주간 캘린더 — 이번 주 달성 현황을 한눈에
• 흐름 카운터 — 연속 달성 일수 추적
• 축하 애니메이션 — 모든 습관 완료 시 컨페티
• 알림 — 습관별 리마인더
• 위젯 — 홈 화면에서 바로 확인
• 다크/라이트 모드 — 취향에 맞게 선택

🔒 개인정보
모든 데이터는 기기에만 저장됩니다. 서버 없음. 계정 없음. 수집 없음.

많으면 지치고, 적으면 지켜져요.
오늘 심은 작은 습관이 내일의 나를 만듭니다.
```

### 앱 아이콘
- 파일: `assets/icon.png`
- 요구사항: 512×512 PNG, 알파 채널 없음, 원본 그대로 사용

### 피처 그래픽 (필수)
- 파일: `docs/play-assets/feature-graphic.html` → 브라우저 캡처 → 1024×500 PNG
- 저장 위치: `docs/play-assets/feature-graphic.png`

### 스크린샷
⚠ **주의**: 기존 `assets/screenshots/final/*.png`는 1242×2688 (2.16:1 비율)로 Play Store 제한(최대 2:1)을 초과. 재생성 필요.
- 권장 해상도: **1080×1920** (정확히 2:1 → 9:16 비율)
- 최소 2장, 최대 8장
- HTML 재활용 가능: `assets/screenshots/final/0X_*.html`을 1080×1920 뷰포트로 열고 재캡처

## 6. 폐쇄 테스트 시작 체크리스트

- [ ] 신원 확인 완료 (Google 검토 중)
- [ ] 앱 만들기 → 패키지명 `com.qlemql.minimalhabittracker` 등록
- [ ] `eas build --platform android --profile production` 으로 AAB 빌드
- [ ] 폐쇄 테스트 트랙 생성 → AAB 업로드
- [ ] 테스터 20명 이메일 리스트 등록
- [ ] 테스터에게 Opt-in URL 전달
- [ ] **14일 연속 테스트 + 테스터당 최소 1회 실행** 완료
- [ ] 프로덕션 심사 요청
