# 게이미피케이션: 해금 시스템 설계 (초안)

> 상태: 설계 대기 (v1.1 이후 구현 예정)

## 컨셉
흐름(연속 달성) N일 달성 시 새로운 아이콘/색상이 해금되는 보상 시스템.
"사야 하는" 게 아니라 "노력으로 얻는" 느낌.

## 해금 테이블 (초안)

| 흐름 달성 | 보상 | 카테고리 |
|-----------|------|----------|
| 7일 | 색상 2개 | 기본 확장 |
| 14일 | 아이콘 4개 | 기본 확장 |
| 30일 | 프리미엄 색상 3개 | 달성감 |
| 50일 | 프리미엄 아이콘 4개 | 달성감 |
| 66일 | 스페셜 테마 | 습관 완성 기념 (학술 근거) |

## 구현 방향
- `habitStore`에 `unlockedRewards: string[]` 추가
- 흐름 일수 변경 시 자동 체크 → 해금 시 축하 토스트
- 습관 추가/수정 화면에서 잠긴 항목은 🔒 + "흐름 N일에 해금" 표시
- 기본 제공: 12개 아이콘 + 8개 색상
- 해금 추가: 아이콘 8개 + 색상 5개

## 헌법 체크
- 규칙 2 (5초): 해금은 자동, 사용자 조작 없음 ✅
- 규칙 6 (설정 없음): 해금 시스템에 설정/토글 없음 ✅
- 규칙 7 (구독 없음): 무료 보상, 결제 불필요 ✅
