# 브랜딩 가이드 — 새싹 아이콘 기반 통합 정리

> 확정된 아이콘: 따뜻한 크림 배경 + 녹색 새싹(잎 3장) + 체크마크
> 핵심 메타포: 성장/자기계발, 작은 시작이 큰 변화로

---

## 1. 스플래시 화면

### 추천안: 크림 배경 (#FFF8F0)

| 항목 | 현재 | 추천 |
|------|------|------|
| 배경색 | #0D0D0D (다크) | **#FFF8F0 (크림)** |
| 이미지 | 아이콘 동일 | 새싹 아이콘 (배경 원 제거, 새싹+체크만) |
| 앱 이름 | 없음 | 아이콘 아래 앱 이름 표시 |
| 슬로건 | 없음 | 앱 이름 아래 태그라인 1줄 |

**크림 배경을 추천하는 이유:**
- 아이콘의 따뜻한 크림 톤과 일관성 유지
- 앱 설치 → 첫 실행 시 아이콘과 동일한 인상을 줌 (브랜드 연속성)
- 한국 MZ 감성 앱들(텍스트리, 그로우, 포레스트 등)이 밝은 톤 스플래시 사용 추세

**다크(#0D0D0D) 유지 시 고려사항:**
- 앱 내부 UI가 다크 모드 기본이므로 전환이 자연스러움
- 단, 크림 아이콘 → 다크 스플래시 → 다크 UI 흐름에서 아이콘만 이질적으로 느껴질 수 있음

**절충안:** 스플래시는 크림(#FFF8F0), 앱 진입 후 다크 UI로 자연스럽게 전환. 많은 앱이 이 패턴을 사용함(카카오톡 노란 스플래시 → 흰색 UI 등).

### 스플래시 레이아웃 구성
```
┌─────────────────────┐
│                     │
│                     │
│      [새싹 아이콘]    │  ← 배경 원 제거, 새싹+체크만 (96~120pt)
│                     │
│      싹(Ssak)       │  ← 앱 이름 (24pt, #2D5016)
│   작은 습관이 자라는 곳  │  ← 태그라인 (14pt, #8B7355)
│                     │
│                     │
└─────────────────────┘
배경: #FFF8F0
```

## 2. 앱 이름

### 현재 문제점
- "Minimal Habit Tracker" — 길고 (22자), 홈 화면에서 잘림, 검색성 낮음
- "셋(Set)" — 3개 메타포와는 맞지만, 새싹/성장 아이콘과 연결 약함

### 추천안

| 순위 | 이름 | 홈 화면 표시 | 장점 | 단점 |
|------|------|-------------|------|------|
| **1** | **싹 (Ssak)** | 싹 | 아이콘과 완벽 일치, 1글자로 기억 쉬움, "싹이 트다" 관용어 활용 | 단독 검색 시 다른 결과 섞일 수 있음 |
| 2 | 새싹 (Saessak) | 새싹 | 직관적, 성장 메타포 명확 | 기존 앱/브랜드와 겹칠 가능성 |
| 3 | 싹트다 (Ssak-teuda) | 싹트다 | "습관이 싹트다" 문맥 완성, 차별화 | 3글자로 다소 길 수 있음 |
| 4 | 셋싹 (Set-ssak) | 셋싹 | 3개 + 새싹 결합, 팀 제안 계승 | 조어라 의미 즉시 전달 어려움 |

### 최종 추천: **싹 (Ssak)**

**근거:**
- 아이콘의 새싹 심볼과 1:1 대응
- 홈 화면 1글자 표시 — 미니멀 앱 정체성과 일치
- "습관의 싹을 틔우다" 스토리텔링 가능
- 앱스토어 검색: 부제로 보완 가능

### 부제 (앱스토어 Subtitle, 30자 이내)
- **"딱 3개, 작은 습관이 자라는 곳"** (15자)
- 대안: "매일 3개, 습관이 싹트다" (12자)

### 앱스토어 전체 이름
```
싹 - 딱 3개, 작은 습관이 자라는 곳
```

## 3. 태그라인/슬로건 통일

### 마스터 태그라인
> **"작은 습관이 자라는 곳"**

이유:
- 새싹(작은 것) → 성장(자라는) 메타포를 직접 표현
- "곳"이라는 단어로 앱이 공간/환경임을 암시
- 짧고 부드러운 톤, MZ 감성 부합

### 용도별 변형

| 용도 | 문구 | 글자 수 |
|------|------|---------|
| **마스터 태그라인** | 작은 습관이 자라는 곳 | 10자 |
| **앱스토어 부제** | 딱 3개, 작은 습관이 자라는 곳 | 15자 |
| **스플래시 화면** | 작은 습관이 자라는 곳 | 10자 |
| **스토어 첫 줄** | 매일 3개, 습관의 싹을 틔워보세요. | 17자 |
| **SNS/마케팅 짧은 버전** | 싹부터 틔워봐 🌱 | 7자 |
| **온보딩 화면** | 딱 3개면 충분해요. 나머지는 싹이 알아서 키워줄게요. | 26자 |

### 기존 문구 정리

| 기존 문구 | 처리 |
|-----------|------|
| "딱 3개만 집중하세요" | → 스토어 설명 본문에서 활용 |
| "적을수록 지켜진다" | → 폐기 (새싹/성장 메타포와 불일치) |
| "많으면 지치고, 적으면 지켜져요" | → "많으면 지치고, 3개면 자라요"로 변형 후 스토어 설명에 활용 |

▸ 앱을 열고 탭 한 번으로 체크. 5초면 끝.

### 키워드 (100자 이내)
```
습관,트래커,루틴,스트릭,미니멀,자기관리,목표,체크리스트,성장,새싹,3개,건강,동기부여
```
변경사항: "성장", "새싹", "3개", "동기부여" 추가

### 카테고리 추천: **건강 및 피트니스 (Health & Fitness)**

| 기준 | 건강 및 피트니스 | 생산성 |
|------|----------------|--------|
| 경쟁 강도 | 높음 (대형 앱 다수) | 중간 |
| 사용자 의도 매칭 | 습관 = 건강 관리로 인식 | 습관 ≠ 업무 생산성 |
| 검색 유입 | "습관 트래커" 검색 시 이 카테고리에서 노출 | 상대적 노출 낮음 |
| 새싹/성장 메타포 적합성 | 자기계발/건강 성장과 자연스러움 | 약간 이질적 |

**결론:** 건강 및 피트니스 유지. 습관 앱 사용자의 검색 패턴과 일치하며, 새싹/성장 브랜딩과도 자연스럽게 어울림.

### 스크린샷 6장 구성안

모든 스크린샷은 크림(#FFF8F0) 배경 프레임 + 녹색 포인트 컬러 통일

| 순서 | 내용 | 상단 카피 | 하단 보조 |
|------|------|----------|----------|
| 1 | 메인 화면 (습관 3개 체크) | **딱 3개면 충분해요** | 탭 한 번으로 끝 |
| 2 | 체크 완료 애니메이션 | **오늘도 한 뿌리** | 매일 조금씩 자라는 중 |
| 3 | 주간 캘린더 뷰 | **자라난 기록, 한눈에** | 주간 달성률 확인 |
| 4 | 스트릭 화면 | **연속 21일 달성!** | 꾸준함이 습관을 만듭니다 |
| 5 | 다크 모드 전체 화면 | **밤에도 편안하게** | 눈 편한 다크 모드 기본 |
| 6 | 위젯 화면 (홈스크린) | **앱을 열지 않아도** | 위젯으로 바로 체크 (Pro) |

**스크린샷 디자인 톤:**
- 배경 프레임: #FFF8F0 (크림)
- 카피 텍스트: #2D5016 (짙은 녹색)
- 보조 텍스트: #8B7355 (따뜻한 브라운)
- 포인트 요소: 작은 새싹/잎 일러스트를 코너에 배치
- 기기 목업: iPhone 15 Pro, 라운드 코너

## 5. 컬러 시스템 (브랜딩 통일)

아이콘에서 파생된 브랜드 컬러:

| 용도 | 색상 | HEX | 사용처 |
|------|------|-----|--------|
| 크림 배경 | 따뜻한 크림 | #FFF8F0 | 스플래시, 스크린샷 프레임, 스토어 배너 |
| 메인 그린 | 새싹 녹색 | #4CAF50 | 체크마크, CTA 버튼, 포인트 |
| 짙은 그린 | 잎사귀 | #2D5016 | 제목 텍스트, 강조 |
| 따뜻한 브라운 | 흙/줄기 | #8B7355 | 보조 텍스트, 캡션 |
| 다크 배경 | 앱 내부 | #0D0D0D | 앱 UI 배경 (기존 유지) |

> 참고: 앱 내부 UI는 기존 다크 모드(#0D0D0D) 유지. 크림/그린 컬러는 브랜딩 자산(스토어, 스플래시, 마케팅)에만 사용.
