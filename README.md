## 왜 만들었나요?

기존 습관 앱들은 너무 복잡합니다. 습관을 무제한으로 추가할 수 있고, 설정할 것이 많고, 결국 앱 자체를 안 열게 됩니다.

Minimal Habit Tracker는 **의도적으로 습관을 3개로 제한**합니다. 정말 중요한 것만 골라서, 매일 5초 만에 체크하고 끝내세요.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **딱 3개 습관** | 의도적 제약. 적을수록 더 잘 지킵니다 |
| **원탭 체크** | 앱 열고 탭 한 번이면 끝. 5초면 충분합니다 |
| **스트릭** | 연속 달성 일수가 쌓이면 놓치기 싫어집니다 |
| **주간 캘린더** | 이번 주 달성 현황을 한눈에 확인 |
| **알림** | 습관별 리마인더로 깜빡하지 않게 |
| **다크 모드** | 아침에도 밤에도 눈이 편안하게 |

## 사용 방법

### 처음 시작할 때
1. 앱을 열면 추천 습관 목록이 나옵니다
2. 원하는 습관을 골라주세요 (최대 3개)
3. 직접 입력도 가능합니다
4. "시작" 버튼을 누르면 끝!

### 매일 할 일
1. 앱을 열고
2. 완료한 습관을 탭해서 체크하고
3. 끝. 정말 이게 다입니다.

### 추가 기능
- **주간 캘린더**: 날짜별로 색상 점이 채워지는 걸 구경하세요
- **스트릭**: 연속 달성 일수가 올라가는 재미가 있습니다
- **테마**: 설정에서 다크/라이트/시스템 모드를 선택할 수 있습니다
